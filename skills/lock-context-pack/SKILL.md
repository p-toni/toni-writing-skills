# LOCK — Context Pack Builder

## Purpose
Freeze meaning and intent before any prose is written.

This is the anti-drift anchor.

## Instructions

Create a CONTEXT PACK from the conversation so far.

Rules:
- Do not improve prose
- Do not add new ideas
- Do not invent data
- If something is missing, write [MISSING: …]
- Be explicit and concrete

Output exactly:

1) One-line decision/ask:
2) Recommendation (3–6 bullets):
3) Why now (3 bullets):
4) Constraints (people / time / dependencies / political):
5) Assumptions (must be true):
6) Options considered (2–3) + tradeoffs:
7) Risks (top 3) + mitigations:
8) Plan (milestones + owners + dependencies):
9) Success metrics:
10) Kill/pivot criteria:
11) Open questions (for reviewers):

12) FINGERPRINT (choose one: Full | Lite | None | Alternative)
- If Full:
  - My take:
  - The core tension is:
  - Constraint that matters:
  - We stop if:
  - What would change my mind:
- If Lite:
  - My take:
  - The core tension is:
- If Alternative:
  - Use the template explicitly provided by the user

## Hard Constraints

- No drafting
- No interpretation
- No stylistic smoothing
- This output is the single source of truth downstream
