# MINIMAL PROOFREAD — Mechanical Edit Only

## Purpose
Improve clarity and correctness without altering thought.

## Instructions

Proofread with minimal edits.

Rules:
- Preserve voice and sentence structure
- Fix grammar and clarity only
- Remove assistant-like filler
- Do not rephrase for style
- Changes must be ≤ 15%

Output exactly:
1) Edited version
2) 3–5 bullets explaining what changed and why

## Hard Constraints

- Never introduce new ideas
- Never smooth uncertainty
- Never rewrite for tone
